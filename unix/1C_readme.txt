Платформа - 1С: Предприятие
Конфигурации - Бухгалтерия, Зарплата и кадры, Торговля

Платформа может работать в двух режимах: Файловом и Клиент-Серверном

Файловый - Файловая база данных: Локально и Как сетевая папка 
			/////////////////////////////////////////
			формат файла *.1CD
			как правило не более чем 10 клиентов, база данных 4 Гб(рекомендации 1С)
			
			////////////////////////////////////////

Клиент-Серверный - База данных SQL: MS SQL, PostgreSQL, DB2, Oracle

Версии платформы под Linuх: для версии с DEB-системами и для версии с RPM-системами
			С декабря 2015 под MAC OS X
			
Конфигурация - это готовое программное решение, созданное на определенной версии платформы.
		Типовые конфигурации - конф. которые разрабатывает сама 1С
		Нетиповые конфигурации - ....

Пользовательский сайт 1С users.v8.1c.ru

Файлы:
		*.1СD - файл содержащий файловую базу данных
		*.DT  - файл выгрузки базы данных. Имеет меньший размер чем 1CD так как данные сжимаются
		*.CF  - файл выгрузки конфигурации, в котором содержится код и структура метаданных
		
Список баз ibases.v8i
		C:\Users\%username%\AppData\Roaming\1C\1CEStart\

Терминальный сервер: Сервер, Лицензии сервера терминалов, RemoteApp		

Система защиты 1С:
		Лицензии: 
		Аппаратные
		///////////////////////////////////////////////////////////////////
			USB-токены, виды:
			однопользовательские - лицензия на одного пользователя
			многопользовательские - лицензии по кол-ву пользователей
			отдельный ключ на сервер 1С
			
			Для сетевых ключей установить HASP LiceseManager
			
			
		///////////////////////////////////////////////////////////////////
		
		Программные (стоят дешевле)
		///////////////////////////////////////////////////////////////////
		Программная лицензия представляет собой файл, который в зашифрованном 
		виде содержит параметры лицензии и характеристики компьютера, для 
		которого данная лицензия получена:
					- сетевое имя
					- модель мат.платы
					- объем оперативной памяти
					- тип и версия BIOS
					- список процессоров и их параметры
					- список сетевых адаптеров и их MAC адреса
					- список жестких дисков и их параметры
				Для платформы под Windws +
					- наименование ОС
					- версия ОС
					- серийный номер ОС
					- дата установки ОС
			!!! При сравнении анализируется только удаление, а не добавление устройств	

		Делятся на: 
			клиентские - однопользовательские, многопользовательские, комбинированные 
			серверные

		Хранятся: C:\ProgramData\1C\licenses\*.lic
		
		//////////////////////////////////////////////////////////////////
		
	Дополнительные параметры запуска - в свойствах базы данных
		\UseHwLicenses+
		\UseHwLicenses- неиспользование аппаратной лицензии
